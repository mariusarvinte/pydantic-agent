# Requirements
Install `uv`: https://docs.astral.sh/uv/getting-started/installation/  
Create an OpenRouter account: https://openrouter.ai/sign-up/  
Generate a OpenRouter API key (no payment required, but will be rate-limited): https://openrouter.ai/workspaces/default/keys

# Running the agent example
Run `uv run main.py`.

That's it!
